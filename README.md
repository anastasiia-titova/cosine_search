# Semantic Search & Clustering Engine
**Задача:** Разработка модуля для семантического поиска технической документации и автоматической тематической группировки.

**Технологии:** SentenceTransformers, Scikit-Learn, NumPy.

**Ключевые моменты реализации:**
- Использование предобученной модели all-MiniLM-L6-v2 для генерации плотных векторных представлений (embeddings).
- Расчет метрики близости через косинусное расстояние, что позволяет находить смысловые соответствия даже при отсутствии лексических совпадений.
- Применение алгоритма K-Means для unsupervised кластеризации данных, что полезно для предварительного анализа больших корпусов текстов перед обучением.
  
**Оптимизация:** Выбор легкой модели эмбеддингов обеспечивает низкую задержку (latency) при инференсе, что критично для real-time систем.
